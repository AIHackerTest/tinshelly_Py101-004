# “猜数字游戏_基础版”说明

1. 程序随机生成一个 20 以内的数字，用户有 10 次机会猜测
  
2. 程序根据用户输入，给予一定提示（大了、小了、正确）
  
3. 猜对或用完 10 次机会，游戏结束


# “猜数字游戏_进阶版”说明

1. 程序内部用0-9生成一个4位数，每个数位上数字不重复，且首位数字不为零，如1942

2. 用户输入4位数进行猜测，程序返回相应提示:
>* 用A表示数字和位置都正确，用B表示数字正确但位置错误
>* 用户猜测后，程序返回A和B的数量
>* 比如2A1B表示两个数字正确且位置正确，1个数字对但位置错误

3. 猜对或用完10次机会，游戏结束
